# Laço Favela 
> Um sistema de coleta e armazenamento de informações que são requeridas pelo G10 favelas para a melhor distribuição de donativos aos beneficiários do projeto..

## 📌 Sobre o Projeto
Nosso projeto tem como foco desenvolver um protótipo de aplicação web que auxilie o G10 favelas na gestão de doações destinada às comunidades atendidas pela iniciativa Presidente de Rua. 

O programa Presidente de Rua tem como finalidade identificar famílias em situação de vulnerabilidade dentro das comunidades. De maneira que, cada favela tenha um "presidente" que visita as famílias e da entrada no processo do sistemas de doações, trazendo responsabilidade e protagonismo para a comunidade local. Atualmente, a coleta dessas informações é realizada por meio de formulários digitais e físicos, porém, esse método se mostra limitado e atrasado, causando dificuldades e riscos para o armazenamento de dados.


## 🚀 O que o software pretende resolver?

Nosso objetivo, é criar uma aplicação web capaz de coletar, armazenar e organizar os dados das famílias atendidas, facilitando o trabalho do presidente de rua. A plataforma deverá permitir o cadastro das famílias beneficiarias, registro e acompanhamento da distribuição de donativos e manutenção de um histórico de atendimentos, garantindo organização, transparência e gerenciamento eficiente das informações. Garantindo assim, o processo de doações mais estruturado, confiável e sustentável, fortalecendo o programa e ampliando sua capacidade de impacto social. 

## 🛠️ Tecnologias Planejadas
Como o projeto está em fase de definição, estas são as tecnologias que pretendo utilizar:
* **Linguagem:** Python 3 / JavaScript
* **Interface:** HTML, CSS e React
* **Banco de Dados:** PostgreSQL
